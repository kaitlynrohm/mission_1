To make a list

CSS code


There are various things that can be styled in a list!


For example


list-style: disc inside; /*This is short hand for all below*/


list-style-type: disc; /*This changes the marker and is the same for both ordered and unordered lists*/


list-style-image: url(image.png); /*This allows you to put an image as the marker*/


List-style-position: inside; /*This dictates where the markers are. Inside the item or outside the item.*/

HTML code


A list is fairly simple to make. Just follow the structure below


<ul> <!--(for unordered) or--> <ol> <!--(for ordered)-->


    <li>Any list items</li>


    <li>More list items</li>


    <li>As many list items as you want!</li>
